## JAR Generation

    $ mvn install


## Run Maven unit tests

    $ mvn test


## JAR Execution

Replace `<numerator>` and `<denominator>` with appropriate numbers.

    $ java -jar org.joeltong-1.0-SNAPSHOT-jar-with-dependencies.jar <numerator> <denominator>

e.g.

    $ java -jar org.joeltong-1.0-SNAPSHOT-jar-with-dependencies.jar 3 97
